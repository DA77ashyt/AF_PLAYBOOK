# AF_PLAYBOOK - 美式橄榄球战术板

一个轻量级的在线美式橄榄球战术绘制工具，纯前端单文件实现，浏览器打开即用，无需注册登录。

## 功能特性

- **球员拖拽放置** — 支持进攻 5 种（QB/RB/WR/TE/OL）+ 防守 4 种（DL/LB/CB/S）位置，每种位置独立配色
- **路线绘制** — 支持直线箭头、折线、平滑曲线（Catmull-Rom 样条），可切换实线/虚线
- **自由绘制** — 按住鼠标拖动绘制球员跑动路线，自动平滑处理
- **画笔模式** — 在球场上自由绘制标注，不绑定球员
- **防守区域绘制** — 支持矩形和多边形区域，可编辑顶点、自定义颜色、添加文字标注
- **文字标注** — 在场地任意位置添加文字，支持自定义字号和颜色
- **LOS 争球线** — 可拖动调整位置，进攻球员自动跟随移动
- **预设阵型模板** — 8 种常用阵型一键加载（I阵型、猎枪、单跑卫、空后场、4-3、3-4、Nickel、Dime）
- **OL 一键上线** — 将场上所有进攻线卫自动排列在争球线上
- **批量操作** — 框选/多选球员，批量移动、调整大小、删除
- **导出 PNG** — 一键导出高清战术图（带时间戳文件名）

## 快速开始

直接在浏览器中打开 `AF_PLAYBOOK.html` 即可使用，无需安装任何依赖。

## 操作模式

| 模式 | 快捷键 | 说明 |
|---|---|---|
| 选择 | `S` | 拖拽球员移动、选中元素、编辑区域顶点 |
| 画路线 | `D` | 为选中球员绘制跑动路线（点击添加路点，双击/Enter 完成） |
| 自由绘制 | `F` | 按住鼠标拖动为选中球员绘制曲线 |
| 区域 | `A` | 绘制防守区域（矩形/多边形，按 `R` 切换） |
| 画笔 | `P` | 自由绘制标注，不绑定球员 |
| 文字 | `T` | 在场地上添加文字标注 |
| 擦除 | `E` | 点击删除球员、路线、区域、文字或笔迹 |

## 快捷键

| 快捷键 | 功能 |
|---|---|
| `L` | 实线/虚线切换 |
| `R` | 矩形/多边形切换（区域模式下） |
| `O` | OL 一键上线 |
| `+` / `-` | 放大/缩小选中球员 |
| `Enter` | 完成路线绘制 / 完成多边形区域 |
| `Escape` | 取消当前绘制 / 取消选中 |
| `Delete` | 删除选中的球员/路线/区域/文字 |
| `Ctrl+点击` | 逐个添加/移除多选球员 |

## 预设阵型

**进攻阵型：** I阵型、猎枪阵型、单跑卫、空后场

**防守阵型：** 4-3 防守、3-4 防守、五分卫（Nickel）、六分卫（Dime）

加载规则：进攻阵型只替换进攻球员，防守阵型只替换防守球员，互不影响。

## 技术栈

- 纯 HTML + CSS + JavaScript，单文件实现
- Canvas 2D 渲染，支持 HiDPI 显示
- 无外部依赖，离线可用

## 运行环境

桌面浏览器（Chrome / Edge / Firefox）
