## 1. 产品简介

美式橄榄球战术板是一款轻量级在线战术绘制工具，无需注册登录，浏览器打开即用。适用于：

- 橄榄球爱好者分析比赛战术
- 教练/球员绘制战术草图
- 球迷讨论和分享战术思路

**运行环境**：Chrome / Edge / Firefox 桌面浏览器，纯前端单文件，离线可用。

## 3. 快速入门

### 第一步：放置球员

从左侧面板拖拽球员图标到球场上，松开鼠标即可放置。

- **进攻球员**（上半部分）：QB（四分卫）、RB（跑卫）、WR（外接手）、TE（近端锋）、OL（进攻线卫）
- **防守球员**（下半部分）：DL（防守线卫）、LB（线卫）、CB（角卫）、S（安全卫）

### 第二步：调整位置

按 `S` 进入选择模式，拖拽已放置的球员移动位置。

### 第三步：绘制路线

1. 点击选中一个球员
2. 按 `D` 切换到画路线模式
3. 在场地上依次点击添加路点
4. 双击或按 `Enter` 完成路线

### 第四步：导出图片

点击顶栏的「导出 PNG」按钮，战术图将保存为带时间戳的 PNG 文件。

## 4. 操作模式详解

工具栏提供 7 种操作模式，通过点击按钮或快捷键切换：

### 4.1 选择模式（S）

默认模式，用于：
- **拖拽球员**移动位置
- **选中球员/路线/区域/文字**进行编辑
- **拖拽文字**移动文字位置
- **鼠标滚轮**缩放选中文字的字号
- **框选球员**批量操作（在空白处按住鼠标拖出选框）
- **Ctrl+点击**逐个添加/移除多选球员
- **拖拽 LOS 线**调整争球线位置
- **拖拽区域顶点**编辑区域形状

### 4.2 画路线模式（D）

为选中的球员绘制跑动路线：
1. 先在选择模式下点击选中一个球员
2. 切换到画路线模式
3. 点击场地添加路点（2 个点为直线，3 个点以上为折线/曲线）
4. 双击或按 `Enter` 完成，按 `Esc` 取消

**路线样式**：
- 点击「实线/虚线」按钮（或按 `L`）切换线条样式
- 点击「折线/曲线」按钮切换是否使用平滑曲线
- 使用「路线颜色」选择器自定义颜色

### 4.3 自由绘制模式（F）

为选中的球员自由绘制曲线：
1. 先选中一个球员
2. 按住鼠标拖动自由绘制
3. 松开鼠标自动完成，曲线会自动平滑处理

> **注意**：自由绘制的路线绑定到球员，删除球员时路线也会被清除。

### 4.4 区域模式（A）

绘制防守区域（zone coverage）：
- **矩形**：按住鼠标拖动绘制
- **多边形**：点击添加顶点，双击或按 `Enter` 闭合
- 按 `R` 或点击「矩形/多边形」按钮切换形状

**区域编辑**（选择模式下）：
- 点击区域选中，可拖拽顶点修改形状
- 双击边线可添加新顶点
- 右键点击顶点可删除顶点（至少保留 3 个）
- 点击已选中的区域后，修改「区域颜色」可更新该区域颜色

**区域标注**：切换到文字模式，点击区域内可添加文字说明。

### 4.5 画笔模式（P）

在球场上自由绘制标注，不绑定球员：
- 按住鼠标拖动自由绘制
- 松开鼠标完成
- 使用「路线颜色」选择器设置画笔颜色
- 画笔笔迹可单独清除（点击「清除笔迹」按钮）

> **与自由绘制(F)的区别**：画笔模式不绑定球员，适合在场上做临时标注和批注。

### 4.6 文字模式（T）

在场地上添加文字标注：
1. 点击场地任意位置，弹出文本输入框
2. 输入文字后按 `Enter` 确认，按 `Esc` 取消
3. 点击已有文字可编辑内容

**文字样式**：
- 使用「字号」输入框控制文字大小（8-40px）
- 使用「路线颜色」选择器控制文字颜色
- 选中文字后修改字号或颜色可实时更新

### 4.7 擦除模式（E）

点击即可删除场上的元素，按以下优先级检测：
1. 文字标注
2. 区域
3. 球员（同时删除其绑定的路线）
4. 路线
5. 画笔笔迹

## 5. 球员管理

### 添加球员

从左侧面板拖拽球员图标到球场上。

### 移动球员

在选择模式下拖拽已放置的球员。

### 批量选择

| 方式 | 操作 |
|---|---|
| 框选 | 在空白处按住鼠标拖出矩形选框，框内球员全部选中 |
| 逐个添加 | 按住 `Ctrl` 点击球员，添加到选中组 / 从选中组移除 |
| 全部取消 | 点击空白处或按 `Esc` |

### 批量操作

选中多个球员后：
- **拖拽移动**：拖动其中一个，所有选中球员一起移动
- **调整大小**：按 `+` 放大、按 `-` 缩小，或在「球员大小」输入框输入数值
- **批量删除**：按 `Delete` 键删除所有选中球员

### 球员大小

- 默认大小为 16px，范围 10-40px
- 选中球员后按 `+` / `-` 调整大小
- 在顶栏「球员大小」输入框直接输入数值
- 球员之间自动保持最小间距，重叠时自动推开

### 删除球员

- 选中后按 `Delete` 键
- 切换到擦除模式点击球员

## 6. 路线绘制

### 画路线（D 模式）

1. 选中球员
2. 切换到画路线模式
3. 点击场地添加路点
4. 双击或 `Enter` 完成

### 自由绘制（F 模式）

1. 选中球员
2. 按住鼠标拖动绘制
3. 松开完成，自动平滑

### 路线样式

| 选项 | 操作 | 快捷键 |
|---|---|---|
| 实线/虚线 | 点击「实线/虚线」按钮 | `L` |
| 折线/曲线 | 点击「折线/曲线」按钮 | — |
| 颜色 | 使用「路线颜色」选择器 | — |

### 路线管理

- 路线绑定到球员，删除球员时路线自动清除
- 选择模式下点击路线可选中，按 `Delete` 删除
- 擦除模式下点击路线直接删除

## 7. 区域绘制

### 绘制矩形

1. 切换到区域模式（`A`）
2. 确认为矩形模式（工具栏显示「矩形」）
3. 按住鼠标从一角拖动到对角
4. 松开完成

### 绘制多边形

1. 切换到区域模式（`A`）
2. 按 `R` 切换到多边形模式（工具栏显示「多边形」）
3. 依次点击添加顶点
4. 双击或按 `Enter` 闭合多边形

### 区域编辑

在选择模式下：
- 点击区域选中
- 拖拽白色顶点修改形状
- 双击边线添加新顶点
- 右键顶点删除（至少保留 3 个）

### 区域颜色与标注

- 使用「区域颜色」选择器设置颜色
- 选中区域后修改颜色可实时更新
- 切换到文字模式点击区域内添加文字标注

## 8. 文字标注

### 添加文字

1. 切换到文字模式（`T`）
2. 点击场地任意位置
3. 在弹出的输入框中输入文字
4. 按 `Enter` 确认

### 编辑文字

- 文字模式下点击已有文字，选中后再次点击进入编辑
- 选择模式下点击文字可选中并拖动位置
- 选中文字后滚动鼠标滚轮可放大/缩小字号

### 文字样式

- **字号**：在顶栏「字号」输入框设置（8-40px）
- **颜色**：使用「路线颜色」选择器设置
- 选中文字后修改字号或颜色可实时更新

### 删除文字

- 编辑时清空内容确认后自动删除
- 擦除模式下点击文字删除
- 选中后按 `Delete` 键

## 9. 画笔自由绘制

画笔模式用于在球场上做自由标注，不绑定球员。

### 使用方法

1. 按 `P` 切换到画笔模式
2. 按住鼠标拖动绘制
3. 松开完成，曲线自动平滑

### 清除笔迹

- 点击顶栏「清除笔迹」按钮清除所有画笔痕迹
- 擦除模式下点击单条笔迹可删除

### 与自由绘制(F)的区别

| | 画笔(P) | 自由绘制(F) |
|---|---|---|
| 绑定球员 | 否 | 是 |
| 删除球员时 | 保留 | 随之删除 |
| 用途 | 临时标注/批注 | 球员跑动路线 |

## 10. LOS 线（争球线）

LOS（Line of Scrimmage）是争球线，橙色虚线，默认位于场地 70% 位置。

### 调整 LOS

在选择模式下，将鼠标移到 LOS 线附近（光标变为拖拽样式），按住拖动即可移动。

### 联动规则

- 拖动 LOS 时，所有**进攻球员**的 Y 坐标跟随移动
- **防守球员**不受影响
- 加载进攻阵型时，位置会根据当前 LOS 偏移
- 加载防守阵型时，位置基于当前 LOS 动态计算：
  - DL（防守线卫）：LOS 上方 10% 处
  - LB（线卫）：LOS 上方 20% 处
  - CB（角卫）：LOS 上方 30% 处
  - S（安全卫）：LOS 上方 40% 处

### OL 一键上线

点击「OL上线」按钮（或按 `O`），将场上所有 OL 球员排列在 LOS 线上，以场地中心为基准紧密排列。

## 11. 预设阵型

左侧面板底部提供预设阵型模板，点击即可一键加载。

### 进攻阵型

| 阵型 | 说明 | 人数 |
|---|---|---|
| I阵型 | I-Formation，经典双跑卫 | 11人 |
| 猎枪阵型 | Shotgun，QB 站在后方 | 11人 |
| 单跑卫 | Singleback，单 RB | 11人 |
| 空后场 | Empty，无 RB | 11人 |

### 防守阵型

| 阵型 | 说明 | 人数 |
|---|---|---|
| 4-3 防守 | 4 DL + 3 LB | 12人 |
| 3-4 防守 | 3 DL + 4 LB | 13人 |
| 五分卫 | Nickel，额外 CB | 13人 |
| 六分卫 | Dime，两个额外 CB | 14人 |

**加载规则**：
- 进攻阵型只替换进攻球员，防守阵型只替换防守球员，互不影响
- 加载阵型时会清除被替换球员的路线

## 12. 导出与清空

### 导出 PNG

点击「导出 PNG」按钮，将当前战术图导出为高清 PNG 图片：
- 文件名格式：`tactics_YYYYMMDD_HHMMSS.png`
- 导出时自动隐藏选中状态的高亮框

### 清除笔迹

点击「清除笔迹」按钮，清除所有画笔模式绘制的笔迹（带确认弹框）。

### 清空

点击「清空」按钮，清除场上所有元素（球员、路线、区域、文字、笔迹），带确认弹框。

## 13. 快捷键一览

### 模式切换

| 快捷键 | 功能 |
|---|---|
| `S` | 选择模式 |
| `D` | 画路线模式 |
| `F` | 自由绘制模式 |
| `A` | 区域模式 |
| `P` | 画笔模式 |
| `T` | 文字模式 |
| `E` | 擦除模式 |

### 操作快捷键

| 快捷键 | 功能 |
|---|---|
| `L` | 实线/虚线切换 |
| `R` | 矩形/多边形切换（区域模式下） |
| `O` | OL 一键上线 |
| `+` / `-` | 放大/缩小选中球员 |
| `Enter` | 完成路线绘制 / 完成多边形区域 |

| `Ctrl+点击` | 逐个添加/移除多选球员 |

> **注意**：当文字输入框激活时，所有快捷键自动禁用，避免误触。
