# 美式橄榄球战术板 — 产品需求文档 (PRD)

## 1. 产品定位

一个轻量级的在线美式橄榄球战术绘制工具，面向个人分析场景，无需注册/登录，打开即用。

## 2. 目标用户

- 橄榄球爱好者，用于分析比赛战术
- 教练/球员，用于快速画战术草图
- 球迷，用于讨论和分享战术思路

## 3. 核心功能

### 3.1 球员拖拽放置

| 项目 | 说明 |
|---|---|
| 球员类型 | 进攻 5 种(QB/RB/WR/TE/OL) + 防守 4 种(DL/LB/CB/S) |
| 操作方式 | 从左侧面板拖拽球员到球场上任意位置 |
| 场上调整 | 选择模式下可拖拽已放置的球员移动位置 |
| 视觉反馈 | 每种位置有独立颜色，选中时高亮白边 |

### 3.2 路线绘制

| 项目 | 说明 |
|---|---|
| 绘制方式 | 选中球员 → 切换画路线模式 → 点击场地添加路点 → 双击完成 |
| 路线类型 | 支持直线箭头(2点)和曲线(3+路点，自动平滑) |
| 路线归属 | 每条路线绑定到一个球员，删除球员时自动清除路线 |
| 颜色选择 | 支持自定义路线颜色 |
| 取消操作 | Esc 键取消当前绘制 |
| 预览 | 绘制中实时显示半透明虚线预览 |

### 3.3 预设阵型模板

| 类别 | 阵型 | 人数 |
|---|---|---|
| 进攻 | I-Formation (I阵型) | 11人：QB/FB/HB + 2WR + TE + 5OL |
| 进攻 | Shotgun (猎枪阵型) | 11人：QB + RB + 3WR + TE + 5OL |
| 进攻 | Singleback (单跑卫) | 11人：QB + RB + 3WR + TE + 5OL |
| 进攻 | Empty (空后场) | 11人：QB + 4WR + TE + 5OL |
| 防守 | 4-3 Defense | 11人：4DL + 3LB + 2CB + 2S |
| 防守 | 3-4 Defense | 11人：3DL + 4LB + 2CB + 2S |
| 防守 | Nickel (五分卫) | 11人：4DL + 2LB + 3CB + 2S |
| 防守 | Dime (六分卫) | 11人：4DL + 1LB + 4CB + 2S |

**加载规则**：进攻阵型只替换进攻球员，防守阵型只替换防守球员，互不影响。

### 3.4 导出图片

- 一键导出全场战术图为 PNG
- 2x 像素比，高清输出
- 文件名带时间戳

## 4. 操作模式

| 模式 | 快捷键 | 功能 |
|---|---|---|
| 选择 | S | 拖拽球员移动位置、选中球员/路线 |
| 画路线 | D | 为选中球员绘制跑动路线 |
| 擦除 | E | 点击球员或路线直接删除 |

其他快捷键：`Delete` 删除选中项，`Esc` 取消绘制/取消选中。

## 6. 非功能需求

- **平台**：桌面浏览器（Chrome/Edge/Firefox）
- **离线可用**：纯前端，无后端依赖
- **性能**：支持同时 22+ 球员 + 10+ 路线，流畅不掉帧
- **无需安装**：浏览器直接打开

## 7. 不包含的功能 (MVP 范围外)

- 账号系统 / 云端存储
- 战术动画播放
- 多人协作 / 分享链接
- 移动端适配
- 战术历史记录 / 撤销重做
