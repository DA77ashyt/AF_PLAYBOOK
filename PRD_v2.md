# 美式橄榄球战术板 — 产品需求文档 (PRD v2)

> 本文档基于实际已实现功能更新，原版 PRD 保留于 `PRD.md`。

## 1. 产品定位

一个轻量级的在线美式橄榄球战术绘制工具，面向个人分析场景，无需注册/登录，打开即用。

## 2. 目标用户

- 橄榄球爱好者，用于分析比赛战术
- 教练/球员，用于快速画战术草图
- 球迷，用于讨论和分享战术思路

## 3. 核心功能

### 3.1 球员拖拽放置

| 项目 | 说明 |
|---|---|
| 球员类型 | 进攻 5 种(QB/RB/WR/TE/OL) + 防守 4 种(DL/LB/CB/S) |
| 操作方式 | 从左侧面板拖拽球员到球场上任意位置 |
| 场上调整 | 选择模式下可拖拽已放置的球员移动位置 |
| 视觉反馈 | 每种位置有独立颜色，选中时高亮白边 |
| 碰撞检测 | 球员之间自动保持最小间距(36px)，重叠时自动推开 |

### 3.2 路线绘制

| 项目 | 说明 |
|---|---|
| 绘制方式 | 选中球员 → 切换画路线模式 → 点击场地添加路点 → 双击/Enter完成 |
| 路线类型 | 支持直线箭头(2点)、折线(3+路点)和曲线(自动平滑) |
| 线条样式 | 支持实线/虚线切换(L键) |
| 路线归属 | 每条路线绑定到一个球员，删除球员时自动清除路线 |
| 颜色选择 | 支持自定义路线颜色 |
| 取消操作 | Esc 键取消当前绘制 |
| 预览 | 绘制中实时显示半透明虚线预览 |

### 3.3 画笔模式（自由绘制）

| 项目 | 说明 |
|---|---|
| 绘制方式 | 选中球员 → 按住鼠标拖动自由绘制曲线 |
| 平滑处理 | 自动使用 Douglas-Peucker 算法简化点数，Catmull-Rom 样条平滑曲线 |
| 线条样式 | 继承当前实线/虚线设置 |
| 颜色 | 继承当前路线颜色 |

### 3.4 防守区域绘制

| 项目 | 说明 |
|---|---|
| 区域形状 | 矩形（拖动绘制）和多边形（点击添加顶点，双击/Enter完成） |
| 颜色自定义 | 支持独立的区域颜色选择器 |
| 区域标签 | 可在区域内添加文字说明 |
| 编辑功能 | 选择模式下可拖拽顶点修改形状，双击边添加顶点，右键删除顶点 |
| 颜色编辑 | 选择模式下点击区域，颜色选择器同步到该区域颜色，修改即更新 |

### 3.5 文字标注

| 项目 | 说明 |
|---|---|
| 添加方式 | 文字模式下点击场地任意位置，弹出文本输入框 |
| 编辑方式 | 再次点击已有文字可编辑内容 |
| 字号控制 | 支持 8-40px 字号自定义 |
| 颜色控制 | 支持自定义文字颜色（使用路线颜色选择器） |
| 删除方式 | 编辑时清空内容确认后删除，或擦除模式下点击删除 |

### 3.6 LOS 线（争球线）

| 项目 | 说明 |
|---|---|
| 默认位置 | y=70%（进攻方在下方，向上进攻） |
| 拖动调整 | 选择模式下可拖动 LOS 线 |
| 联动规则 | 拖动 LOS 时所有进攻球员 Y 坐标跟随移动 |
| 防守联动 | 防守阵型加载时自动基于当前 LOS 定位（DL在LOS上方10%处） |

### 3.7 OL 一键上线

| 项目 | 说明 |
|---|---|
| 功能 | 一键将场上所有 OL 球员排列在 LOS 线上 |
| 排列方式 | 以场地中心(x=50%)为基准，紧密排列，间距6% |
| 快捷键 | O 键 |

### 3.8 预设阵型模板

| 类别 | 阵型 | 人数 |
|---|---|---|
| 进攻 | I-Formation (I阵型) | 11人：QB/FB/HB + 2WR + TE + 5OL |
| 进攻 | Shotgun (猎枪阵型) | 11人：QB + RB + 3WR + TE + 5OL |
| 进攻 | Singleback (单跑卫) | 11人：QB + RB + 3WR + TE + 5OL |
| 进攻 | Empty (空后场) | 11人：QB + 4WR + TE + 5OL |
| 防守 | 4-3 Defense | 12人：5DL + 3LB + 2CB + 2S |
| 防守 | 3-4 Defense | 13人：5DL + 4LB + 2CB + 2S |
| 防守 | Nickel (五分卫) | 13人：5DL + 3LB + 3CB + 2S |
| 防守 | Dime (六分卫) | 14人：5DL + 3LB + 4CB + 2S |

**加载规则**：
- 进攻阵型只替换进攻球员，防守阵型只替换防守球员，互不影响
- 进攻阵型位置会根据当前 LOS 偏移
- 防守阵型位置基于当前 LOS 动态计算（DL=LOS-10%, LB=LOS-20%, CB=LOS-30%, S=LOS-40%）

### 3.9 导出图片

- 一键导出全场战术图为 PNG
- 文件名带时间戳
- 导出时自动隐藏选中状态的高亮框

## 4. 操作模式

| 模式 | 快捷键 | 功能 |
|---|---|---|
| 选择 | S | 拖拽球员/LOS线移动位置、选中球员/路线/区域/文字、编辑区域顶点 |
| 画路线 | D | 为选中球员绘制跑动路线 |
| 画笔 | F | 为选中球员自由绘制曲线 |
| 区域 | A | 绘制防守区域（矩形/多边形） |
| 文字 | T | 添加/编辑文字标注 |
| 擦除 | E | 点击球员、路线、区域或文字直接删除 |

### 4.1 快捷键一览

| 快捷键 | 功能 |
|---|---|
| S / D / F / A / T / E | 切换操作模式 |
| L | 实线/虚线切换 |
| R | 矩形/多边形切换（区域模式下） |
| O | OL 一键上线 |
| Enter | 完成路线绘制 / 完成多边形区域 |
| Escape | 取消当前绘制 / 取消选中 |
| Delete | 删除选中的球员/路线/区域/文字 |

**注意**：当文字输入框激活时，所有快捷键自动禁用，避免误触。

## 6. 数据结构

### 6.1 球员
```javascript
{ id, type, x, y, isOffense, color }
```
- x, y: 百分比坐标 (0-100)

### 6.2 路线
```javascript
{ id, playerId, points: [{x,y}], color, dashed, useCurve }
```
- 每条路线绑定到一个球员

### 6.3 区域
```javascript
{ id, shape: 'rect'|'polygon', points: [{x,y}], color, label, fontSize }
```
- rect 形状存储 2 个对角点
- polygon 形状存储 N 个顶点

### 6.4 文字
```javascript
{ id, x, y, text, fontSize, color }
```

## 7. 场地坐标系

- **X 轴**：0% = 左边线，100% = 右边线
- **Y 轴**：0% = 防守端区（上方），100% = 进攻端区（下方）
- **进攻方向**：从下方向上方进攻（Y 递减）
- **LOS 线**：默认位于 y=70%

## 8. 非功能需求

- **平台**：桌面浏览器（Chrome/Edge/Firefox）
- **离线可用**：纯前端，无后端依赖，单文件 HTML
- **性能**：支持同时 22+ 球员 + 10+ 路线 + 多个区域，流畅不掉帧
- **无需安装**：浏览器直接打开
- **渲染**：Canvas 2D，支持 HiDPI 显示

## 9. 不包含的功能 (当前版本外)

- 账号系统 / 云端存储
- 战术动画播放
- 多人协作 / 分享链接
- 移动端适配
- 战术历史记录 / 撤销重做
